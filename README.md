## React and Redux training
#### Course by Cory House at Pluralsight

Linting the project:
```
npm run lint
npm run lint:watch
```

Bundling with webpack and serving the app using express:
```
npm run open:src
```

Unit testing:
```
npm test
npm run test:watch
```

All together in parallel (npm-run-all --parallel):
```
npm start
```
